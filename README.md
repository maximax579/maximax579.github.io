# Grafik website

Grafik is a simple program that lets you visualise even the most complicated polynomial functions on your computer screen. This  website is dedicated to project "Grafik".

## Technologies

Libraries used to build website:
* Bootstrap 4.3.1
* jQuery 3.4.1
* parralax.js 1.5.0
